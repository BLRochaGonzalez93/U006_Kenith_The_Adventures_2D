# Kenith: The Adventures 2D Visual Pack

[English](README.en.md) | [Español](README.md)

Planned images for the **Kenith: The Adventures 2D** project.

This directory is prepared to include the project's visual assets in original and web-optimized versions.

## Recommended usage

- `keniththeadventures-logo.webp` for project identity.
- `keniththeadventures-cover.webp` for Open Graph, cover and social sharing.
- `keniththeadventures-banner.webp` for the portfolio website.
- Thumbnails for cards, documentation, README files or technical previews.
