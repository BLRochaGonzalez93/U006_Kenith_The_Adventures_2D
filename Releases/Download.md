- Protagonista principal: Kenith.

- Recolección de recursos, pickups y coleccionables.
- Inventario, equipamiento y sistema de crafteo.
- Sistema de vida, daño, niveles, experiencia y estadísticas.
- Misiones, tutorial, diálogos, narrativa, UI, sonido y música.
- Build jugable para Windows.

## Cómo ejecutar o instalar

1. Abrir la página de GitHub Releases enlazada arriba.
2. Descargar el archivo de la build disponible en la sección **Assets**.
3. Extraer el `.zip` en una carpeta local.
4. Ejecutar el archivo `.exe` del juego.
5. Mantener el `.exe` dentro de la carpeta extraída, junto con los archivos internos generados por Unity.
